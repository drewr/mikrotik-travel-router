# Agent Instructions

## Commit style

Use [Conventional Commits](https://www.conventionalcommits.org/):

```
<type>[optional scope]: <description>
```

Common types: `feat`, `fix`, `docs`, `refactor`, `test`, `chore`

Examples:
- `feat: add LAN_SUBNET parameter`
- `fix: use :do on-error for root user creation`
- `docs: rewrite README around generator workflow`
- `test: add exit_ipv4_and_ipv6_survive_rerun integration test`
- `refactor: replace inline DHCP script with named system script`
